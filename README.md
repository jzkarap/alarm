# Alarm

<img src="https://github.com/jzkarap/alarm/blob/master/alarm.gif?raw=true">

Vanilla JavaScript alarm that runs in a browser.

Features:
- Set time in predetermined increments
- Start/pause timer
- Blares evacuation alarm when timer is up

To add:
